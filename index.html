Project

Music auto-tagging

Collaborators: Machine Learning Research Team, KKBOX Inc
Advisor: Dr. Yi-Hsuan Yang

Music auto-tagging project can be divided into three sub-projects: instrument recognition, genre tagging, and emotion tagging.

Instrument Recognition focuses on polyphonic frame-level instrument tagging. The model will predict the presence of instruments in each time step so that we can know the starting time and ending time of each instrument. We proposed to use pitch to guide the learning of instrument tags, with the premise that doing so informs the model the notes that are active per frame, and also encourages the model to learn relative rates of energy buildup in the harmonic partials of different instruments. The model is trained with MusicNet Dataset. Please visit our demo page for further details.

The second stage of this project is to increase instrument categories. We collected a large-scale synthesized dataset, Musescore Dataset, to train the model. Musescore Dataset contains over 330,000 pieces of songs with their related audio and MIDI pairs. Please visit our demo page to see the result.

Music Genre Tagging focuses on music genre and subgenre taggings. For example, given a song from Charlie Parker, except for telling us the song belongs to jazz, the model will also tell us it belongs to Swing and Bebop. We trained the model on the mix of private dataset and Million Song Dataset(MSD). Please visit our project page for more details.

Music Emotion Tagging is part of the project for music generation. We trained a deep learning model by private dataset for audio emotion tagging (arousal and valence). We transfered the tags to MIDI domain by using Lakh MIDI Dataset(LMD), which contains large-scale MIDI and audio pairs. The MIDI emotion tags will then be used as a condition for conditional music generation.

Multi-pitch Streaming

Advisor: Dr. Yi-Hsuan Yang

Music streaming project is the extension application of the instrument recognition project. We proposed a specialized multitask model to predict pitch, instrument, and pianorolls at the same time. By jointly predicting the labels, the model achieves better performance than other single-task models.The predicted pianorolls can also be used to recognize the instrument that plays each individual note event and achieve multiinstrument transcription. Please see our paper or Github page for more details.

Music Disentanglement

Advisor: Dr. Yi-Hsuan Yang

For some applications we need a feature representation that is not sensitive to the changes in other features. For example, in cover song identification or query by humming we only need pitch information while in singer identification we only need timbre information. As a result, in this music disentanglement project, we proposed a deep auto-encoder model with adversarial training to learn timbre and pitch invariant representations. Please visit our paper for more details.
