Project

Music auto-tagging

Collaborators: Machine Learning Research Team, KKBOX Inc
Advisor: Dr. Yi-Hsuan Yang

The music auto-tagging project can be divided into three sub-projects: instrument recognition, genre tagging and emotion tagging.

Instrument Recognition focuses on polyphonic frame-level instrument tagging, which means the model will predict the presence of instrument in each time step, so that we can know the starting time and ending time of each instrument. We research on using pitch to strengthen the harmonic structure in polyphonic music, which can also benefit instrument recognition result. The model is trained with MusicNet Dataset. Please visit our demo page for further details.

Second stage of this project is to provide large scale training data. We proposed another join-learning model for training large-scale data. The model is trained with Musescore Dataset, a new dataset containing polyphonic audio and midi pairs. (Result Demo)

Music Genre Tagging focuses on music genre and subgenre taggings. For example, given a song from Charlie Parker, except for telling us the song belongs to jazz, the model will also tell us it belongs to Swing and Bebop. We trained the model on the mix of KKBOX private dataset and Million Song Dataset(MSD). Please visit our demo page for more details.

Music Emotion Tagging is part of the project to generate music according to user's emotion. We train a deep learning model for emotion tagging on audio domain, and transfer the tags to MIDI domain by using Lakh MIDI Dataset(LMD), which contains large-scale MIDI audio pairs. The MIDI emotion tags will be used as a condition to generate music.

Music Streaming

Music streaming is the extension application of the instrument recognition project. Since the model has already learned the pitch and instrument information, it can further predict which instrument is playing the note in each time step. The streaming result can also extend to msic auto-transcription.
